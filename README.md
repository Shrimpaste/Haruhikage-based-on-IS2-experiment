# 春日影 - I2S音频播放器 (Haruhikage I2S Player)

基于 ESP32S3 + MAX98357 的《春日影》前奏和弦播放器，通过UART串口控制播放与停止。

## 硬件需求

| 组件 | 型号/规格 |
|------|----------|
| 开发板 | ESP32S3 |
| 音频放大器 | MAX98357 I2S 功放模块 |
| 扬声器 | 4Ω-8Ω 喇叭 (3W-10W) |
| 杜邦线 | 若干 |

## 详细接线表

| ESP32S3 | MAX98357 | 说明 |
|---------|----------|------|
| 3.3V | VCC | 电源供电 |
| GND | GND | 地线 |
| GPIO 5 | BCLK | I2S位时钟 |
| GPIO 6 | LRC (WS) | I2S字选择 |
| GPIO 7 | DIN | I2S数据输入 |
| 3.3V | SD | 模块使能 (可接GPIO控制静音) |
| NC/GND | GAIN | 增益设置 (GND=9dB) |

## UART通信

默认使用 UART0 (USB串口):
- 波特率: 115200
- 数据位: 8
- 停止位: 1
- 校验: 无

### 控制命令

| 命令 | 功能 |
|------|------|
| `p` 或 `P` | 开始播放 |
| `s` 或 `S` | 停止播放 |
| `h` 或 `H` | 显示帮助 |

## 编译与烧录

### 环境要求

- ESP-IDF v5.0 或更高版本
- Python 3.7+

### 编译步骤

```bash
# 设置ESP-IDF环境
. $HOME/esp/esp-idf/export.sh

# 进入项目目录
cd haruhikage_i2s

# 配置项目 (选择ESP32S3)
idf.py set-target esp32s3

# 编译
idf.py build

# 烧录 (替换PORT为你的串口号)
idf.py -p /dev/ttyUSB0 flash

# 查看串口输出
idf.py -p /dev/ttyUSB0 monitor
```

## 技术参数

- **采样率**: 44.1kHz
- **采样位深**: 16-bit
- **声道**: 单声道 (Mono)
- **输出格式**: I2S标准模式 (MSB)

## 音符说明

本程序基于《春日影》简谱前奏部分，B调，6/8拍，速度97 BPM。

播放的音高对应简谱:
- `3 2 1 2` → D#5 C#5 B4 C#5
- `3. 4 3 2.` → D#5. E5 D#5 C#5.

## 注意事项

1. **电源**: MAX98357需要3.3V供电，不能直接用5V
2. **扬声器**: 建议使用4Ω或8Ω扬声器，功率3W以上
3. **增益**: GAIN引脚接GND为9dB增益，接3.3V为3dB，悬空为12dB
4. **SD引脚**: 接低电平可关闭输出，可用于静音控制
5. **长时间播放**: 程序循环播放前奏，可通过UART命令停止
